BANCO DE DADOS:
● Pets: Tabela que armazena informações dos pets (id, nome, espécie, idade,
descrição, status).
● Adotantes: Tabela que armazena informações dos adotantes (id, nome,
email, telefone, endereço).
● Adoções: Tabela que registra as adoções realizadas, com referência ao pet
e adotante (id, pet_id, adotante_id, data_adocao).

estado_adocao = 'adotado' OU 'não-adotado'

BACKEND:
○ O backend será responsável pela lógica do sistema e gestão dos dados. Deve ser
desenvolvido com Node.js, utilizando o framework Express.
○ Utilizar o Prisma ORM para realizar as operações no banco de dados.
○ Implementar operações de CRUD (Create, Read, Update, Delete) para gerenciar as
informações de pets e adotantes.

FRONTEND:
O frontend será desenvolvido com ReactJS, proporcionando uma interface amigável
e dinâmica para o usuário.
O frontend deve:
○ Exibir a lista de pets.
○ Incluir formulários para cadastrar pets e adotantes.
○ Exibir o histórico de adoções e permitir filtragem de pets.
○ Landing page que apresenta o sistema de forma elegante, destacando os
benefícios do abrigo e a facilidade no processo de adoção.
