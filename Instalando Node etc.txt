Instalando/inicializando o Node, parâmetro "-y" para pular as perguntas de configuração e deixar padrão:
npm init -y

Instalando Express:
npm install express

Instalando PostgreSQL para conexão com o banco:
npm install pg

Instalando nodemon para atualização do servidor em tempo real:
npm install nodemon

Instalando o PRISMA ORM:
npm install @prisma/client prisma
npx prisma init

Puxando o banco com PRISMA ORM:
npx prisma db pull

Gerar schema com PRISMA ORM:
npx prisma generate

Usando migrate:
npx prisma migrate dev

Instalando Bcrypt para criptografar senhas:
npm install bcryptjs jsonwebtoken
